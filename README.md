## Installation

Install `slmsuite` from [PyPI](https://pypi.org/project/slmsuite/):

```console
pip install slmsuite
```

Or install the latest version from GitHub:

```console
pip install git+https://github.com/slmsuite/slmsuite
```

Then clone this examples repository:

```console
git clone https://github.com/slmsuite/slmsuite-examples
```
